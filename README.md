A mobile-friendly credit tracking app for high school students to monitor their progress toward graduation, with special support for dual credit courses.

## Features

- **Student Dashboard**: Track credits by category with visual progress indicators
- **Dual Credit Tracking**: Identify which courses count toward Associate vs. Transfer degrees
- **Counselor Portal**: View all students, identify at-risk students, monitor progress
- **PDF Transcript Export**: Generate printable transcripts
- **Smart Notifications**: Alerts for students falling behind

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This app is configured for easy deployment on Vercel:

1. Push to GitHub
2. Connect repository to Vercel
3. Deploy automatically
